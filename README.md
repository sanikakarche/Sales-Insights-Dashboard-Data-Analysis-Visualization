# 📊 Sales Intelligence & Revenue Analysis 
**Data Engineering & BI Implementation | T.E. Computer Engineering**

## 🔍 Project Overview
This project focuses on building an end-to-end Business Intelligence pipeline to analyze sales performance for a hardware distributor. The goal was to transform raw transactional data into a "Single Source of Truth" to identify revenue leaks and market trends.

## 🛠️ Technical Stack
* **Database:** MySQL (Relational Schema & Data Auditing)
* **ETL:** Power Query (Data Cleaning & Currency Normalization)
* **Visualization:** Power BI (Data Modeling & DAX)

## 🚀 Technical Implementation
* **Data Auditing (SQL):** Conducted manual SQL audits to identify data anomalies, such as negative sales amounts and invalid transaction records.
* **Currency Normalization:** Developed a transformation layer to handle multi-currency transactions (USD/INR), ensuring accurate global revenue reporting.
* **Schema Design:** Implemented a **Star Schema** to optimize query performance and dashboard responsiveness.
* **Calculated Measures:** Developed DAX formulas for dynamic KPIs including `Total Revenue`, `Profit Margin %`, and `Market Share`.

## 📂 Repository Structure
* `Atliq_Database_Dump.sql`: Full database schema and raw data.
* `Data_Audit_Queries.sql`: SQL scripts used for initial data validation.
* `Sales_Insights_Final.pbix`: Production-ready Power BI Dashboard.
